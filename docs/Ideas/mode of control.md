# Mode of control (working title)

not every sub can be controlled all the time. Many have families or other obligations.

The system should cater for that with a mode system that defines the deepness of control the AI expects and performs

I would start with the following modes:

- Full supervision

The sub is in full control and the mistress can use the sub at her mercy all the time if she wishes

- Discreet supervision

E.g. familiy and kids are around and only certain toys can be used and no noise can be produced. This mode should be explained by the sub. so mistress can understand what is possible
The sub is in full control but can only perform discreet tasks with certain agreed upon toys and kinks that can be discreetly done. 

- Task mode

Mistress only gives tasks with gracefull fulfillment timers, sub has to do before the timer runs out

- Homeoffice

A special mode of discreet supervision indicating mistress that sub is working and cannot react right away due to meetings and that the tasks and toys have to be discreetly usable

- Vacation

Pauses the training for a certain time period


There should be a possiblity to create a schedule for these modes. There should be a calendar view for it
